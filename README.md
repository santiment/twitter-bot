# twitter-bot
Nodejs service that works as webhook for santiment alerts and reposts to twitter.

## Setup

To setup the service install all node packages:

```bash
$ npm install
```

## Running the service

```bash
$ npm start
```

## Running tests

```bash
$ npm test
```

## Using Docker

Running the server with docker:

```bash
$ npm run docker-dev
```


Running tests with docker:
```bash
$ npm run docker-test
```
